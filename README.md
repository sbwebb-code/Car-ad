# Car Ad Builder – East Tennessee Auto Outlet

![Preview](preview.png)

**Instantly create stunning 16:9 (Facebook) or 1:1 (Instagram) car ads** with festive overlays, dealership branding, and ready-to-post captions — **no design skills needed**.

Perfect for used car dealerships, sales teams, and social media managers.

---

## Features

- Upload any car photo
- Auto-generates **two professional captions** (short + long)
- Live preview in **16:9** (Facebook) or **1:1** (Instagram square)
- One-click **PNG download**
- Fully **customizable** (colors, logo, text, hashtags)
- Works **offline** after first load
- **Zero cost** – runs in any browser

## How to Use

1. **Upload** a high-quality photo of the car
2. **Fill in** the form (most fields pre-filled with your example)
3. **Click "Render Preview"**
4. **Switch** between **16:9** and **1:1** using the buttons
5. **Download** the image
6. **Copy** the short or long caption
7. **Post** to Facebook or Instagram!
